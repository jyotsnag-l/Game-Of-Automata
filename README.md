# 🤖 Game of Automata

A sleek, interactive puzzle game where you step into the role of an Automata Engineer! Your mission: identify and fix logic errors in Finite Automata (DFA/NFA) generated by AI.



## 🎮 How It Works

In each level, you are presented with a **Finite Automaton** that is *almost* correct. The transitions and states are perfect, but the **Final States** (the accepting states) are wrong!

1.  **Analyze the Language:** Read the description of the language the automaton *should* accept (e.g., "Accepts strings ending in '01'").
2.  **Inspect the Graph:** Look at the states and transitions. Follow the paths for various inputs.
3.  **Toggle States:** Click on any state to toggle it between a **Normal State** and a **Final State** (represented by a glowing double circle).
4.  **Solve:** Match the set of final states to the correct logic to win!

## ✨ Key Features

*   **🧠 AI-Powered Puzzles:** Every puzzle is uniquely generated using the **Gemini 2.5 Flash** model, ensuring a fresh challenge every time.
*   **📈 Three Difficulty Levels:**
    *   **Easy:** Simple linear structures with 3 states and a single error.
    *   **Medium:** 4-5 states with cycles and more complex logic.
    *   **Hard:** Subtle errors in branching structures that require deep analysis.
*   **🛠️ Developer Tools:**
    *   **Undo/Reset:** Made a mistake? Step back or start over instantly.
    *   **Hints:** Get a gentle nudge in the right direction if you're stuck.
*   **🎨 Cyberpunk Aesthetic:** A high-contrast "Deep Navy & Amber" theme with blueprint-style grid backgrounds and smooth animations.

## 🚀 Tech Stack

*   **Frontend:** React 19 + TypeScript
*   **Styling:** Tailwind CSS
*   **AI Engine:** Google Gemini API (`@google/genai`)
*   **Build Tool:** Vite
*   **Icons:** Lucide React

## 🛠️ Getting Started

### Prerequisites
*   Node.js installed
*   A Gemini API Key (set as `GEMINI_API_KEY` in your environment)

### Installation

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

3.  **Open the app:**
    Navigate to `http://localhost:3000` in your browser.
